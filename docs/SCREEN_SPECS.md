# Screen Specs

## 1. Home / Today
### Purpose
Quick answer to “What is today’s Panchanga?”

### Sections
- top date hero
- Panchanga summary card
- sunrise/sunset card
- today’s special tithis
- rahukalam / important windows
- CTA to open full day page

## 2. Month Calendar
### Purpose
Visual monthly browse

### Requirements
- month grid
- Gregorian date large
- short tithi label
- festival marker dot / tag
- swipe or arrow navigation
- tap a day to open day detail

### Cell content
- Gregorian date
- primary tithi at sunrise
- 0–2 markers for important special tithis
- current day highlight

## 3. Day Detail
### Purpose
Full Panchanga detail for a selected date

### Sections
- date header
- summary card
- intraday transitions timeline
- special tithis/festivals
- muhurtha / windows
- source-note/raw details toggle (debug/admin only)

### Timeline items
- tithi transition
- nakshatra transition
- yoga transition
- karana transition

## 4. Special Tithis
### Purpose
List and filter special tithis by date/category

### Filters
- festival
- vrata
- ekadashi
- punyadina
- sankramana
- all

## 5. Muhurthas
### Purpose
Browse auspicious dates

### Filters
- vivaha
- upanayana
- vastu
- all

## 6. Search
### Purpose
Global lookup across days and special tithis

### Searchable fields
- date
- special tithi name
- tithi
- nakshatra
- muhurtha type

## 7. Settings
### v1
- theme preference
- about
- data version

### v2
- location
- language
- notification preferences
