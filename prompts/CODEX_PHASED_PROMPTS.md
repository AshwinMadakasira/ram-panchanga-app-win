# Codex phased prompts

## Prompt 1 — Scaffold
Create the Expo TypeScript project structure, install dependencies, configure Expo Router, NativeWind, Reanimated, Zustand, TanStack Query, and SQLite. Add a root README with setup commands.

## Prompt 2 — Database
Implement the schema from `data/schema.sql`, create a database bootstrap module, and create repository functions for month summary, day detail, special tithis, and muhurthas.

## Prompt 3 — Import
Write an import script that reads the extracted Panchanga database and maps it into the app schema. Create a deterministic seed flow for `los-angeles-ca`.

## Prompt 4 — Month calendar
Build the month calendar screen with an attractive grid, current day highlight, tithi label, and special tithi markers. Tapping a day should navigate to `/day/[date]`.

## Prompt 5 — Day detail
Build the day detail screen with:
- summary card
- sunrise/sunset
- transitions timeline
- special tithis list
- muhurtha snippets
- excellent spacing and typography

## Prompt 6 — List screens
Build special tithis, muhurthas, and search screens with filters and clean states.

## Prompt 7 — Polish
Improve performance, empty states, loading states, error handling, and visual consistency. Ensure the app is store-ready in structure.
